# Cardano Ultimate Savings Vault
A production-ready Cardano DApp with Mesh SDK and Plutus V2.

## Key Features
- **Smart Contract**: Haskell Plutus V2 script enforcing savings targets.
- **Frontend**: Next.js 14 with Tailwind CSS.
- **Wallet Connection**: Supports Nami, Eternl, Flint, etc. via Mesh SDK.
- **Vercel Ready**: Optimized for immediate deployment.

## Deployment Instructions (Vercel)
1. Upload the contents of this ZIP to a new GitHub repository.
2. In Vercel, import the repository.
3. Set **Node.js Version** to 18.x or 20.x in Project Settings.
4. No other configuration needed (Next.js will auto-detect everything).
